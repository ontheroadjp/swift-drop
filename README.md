# Swift Drop

DL URL と認証コードでファイルを渡す簡易サービスです。

## Features

- 100MB までのファイルを複数アップロード（1回20ファイルまで）
- 有効期限を日単位で指定可能（1〜30日）
- アップロードごとに DL URL と認証コードを発行
- 受け取り側は URL + 認証コード入力後、対象ファイル一覧から個別またはまとめてダウンロード
- 認証コードはハッシュ化して SQLite に保存

## Setup

```bash
npm install
npm run dev
```

- 開発サーバー: `http://localhost:3000`
- ポート指定: `npm run dev --port 5000`

## Usage

1. `http://localhost:3000` で複数ファイルをアップロード
2. 表示された `DL URL` と `認証コード` を受け取り側へ共有
3. 受け取り側は URL にアクセスし、認証コードを入力して一覧表示
4. 一覧から個別ダウンロード、または選択/全件を ZIP でまとめてダウンロード

## Notes

- 保存先ファイル: `uploads/`
- メタデータ DB: `data/swift_drop.db`
- デフォルト有効期限: 発行から 7 日間
